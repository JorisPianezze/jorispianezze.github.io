Welcome to CROCO / ABL1d coupling’s documentation¶

The purpose of this tutorial is to present the different steps to set up :

  • an idealized simulation with the Atmospheric Boundary Layer 1d (ABL1d) model based on Kilpatrick et al. [2014] case study (Section Run KILPATRICK test cases; see also Kilpatrick et al. [2014], Spall [2007], Ayet and Redelsperger [2019], Lemarié et al. [2021]) ;

  • a real case ABL1d - CROCO based on BENGUELA_LR ktest (Section Run BENGUELA_LR test case).

Description of the ABL1d model can be found in Lemarié et al. [2021]. Same version of the ABL1d model described in this paper has been implemented in CROCO.

Note

This documentation is subject to change, so don’t hesitate to send me your improvements, corrections, etc …
