Welcome to Meso-NH / WW3 / CROCO coupling’s documentation¶

This documentation is intended for Meso-NH users who are interested to learn how to couple it with WW3 and/or CROCO models using the OASIS coupler.

Note

This documentation is subject to change, so don’t hesitate to send me your improvements, corrections, etc …
