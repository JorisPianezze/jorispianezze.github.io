Introduction

This documentation describes how to set up and run a coupled simulation between Meso-NH and WW3 and/or CROCO with the OASIS coupler. It doesn’t contain detailed explanations on the use of Meso-NH, WW3, CROCO or OASIS. You are therefore advised to familiarise yourself with these models before reading this documentation.

Warning

This documentation has been tested with version 5.7.2 of Meso-NH, 2.1.0 of CROCO, 7.14 of WW3 and 5.2 of OASIS.

The developments were carried out as part of an Ocean-Wave-Atmosphere project group initiated by the Centre National de Recherches Météorologiques (CNRM) around the SURFEX surface model [Voldoire et al., 2017, Pianezze et al., 2018].
