Welcome to Meso-NH / WW3 / CROCO coupling's documentation
============================================================


This documentation is intended for `Meso-NH <http://mesonh.aero.obs-mip.fr/>`_ users who are interested to learn how to couple it with `WW3 <https://polar.ncep.noaa.gov/waves/wavewatch>`_ and/or `CROCO <https://www.croco-ocean.org>`_ models using the `OASIS <https://oasis.cerfacs.fr/en/home/>`_ coupler.

.. note::

   This documentation is subject to change, so don't hesitate to `send me <joris.pianezze@cnrs.fr>`_ your improvements, corrections, etc ...
